# 需求说明：账户激活与邀请分佣机制

## 1. 账户激活

| 项目         | 说明                                             |
| ------------ | ------------------------------------------------ |
| **激活费用** | 0.015 BNB                                        |
| **激活性质** | **终身一次性激活**（玩家死亡后复活无需再次激活） |
| **前置条件** | 新用户必须支付激活费后，才能开始签到挖矿         |
| **状态变更** | 未激活 → 已激活（永久）                          |

> [!IMPORTANT]
> 激活状态与游戏内生死状态无关，一经激活，永久有效。

---

## 2. 邀请分佣机制

### 分佣比例

| 层级       | 比例    | 说明           |
| ---------- | ------- | -------------- |
| 一级邀请人 | **33%** | 直接邀请者     |
| 二级邀请人 | **13%** | 邀请者的邀请者 |
| 团队金库   | **54%** | 剩余部分       |

### 分佣逻辑

```
当用户支付 0.015 BNB 激活时：
├─ 有一级+二级邀请人 → 一级得33%，二级得13%，团队得54%
├─ 仅有一级邀请人   → 一级得33%，团队得67%
└─ 无邀请人        → 团队得100%
```

### 金额示例（按0.015 BNB计算）

| 场景       | 一级    | 二级    | 团队    |
| ---------- | ------- | ------- | ------- |
| 完整邀请链 | 0.00495 | 0.00195 | 0.0081  |
| 仅一级     | 0.00495 | -       | 0.01005 |
| 无邀请人   | -       | -       | 0.015   |
